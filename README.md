# Border Radius Generator

A small interactive tool for generating CSS `border-radius` values.  
The project was created as a learning exercise to practice basic JavaScript, DOM manipulation, and responsive CSS.

## About

This is a simple educational project built with plain HTML, CSS, and JavaScript.  
Its purpose is to practice working with JavaScript events, updating styles dynamically, and keeping the code structure clear and readable.

The tool allows users to adjust each corner of a rectangle using sliders and instantly see the result.  
The generated CSS can be copied and used directly in other projects.

---

## Features

- Adjust border-radius for each corner separately
- Live visual preview
- Copy CSS output to clipboard
- Mobile-first layout
- No frameworks or external libraries

---

## Technologies

- HTML5
- CSS3 (mobile-first layout, custom reset)
- Vanilla JavaScript

---

## Usage

Open the `index.html` file in a web browser.
